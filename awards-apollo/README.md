# Awards Service - Apollo/NodeJS

The awards service knows about Science Fiction literature awards like the Hugo and Nebula.

# Setup

 1. Run `npm install`
 2. Add a `.env` file.

# Run a Schema Check

 1. Edit the Makefile to specify your correct graph and variant.
 2. Run `make check`

# Push a Schema Change

 1. Edit the Makefile to specify your correct graph and variant.
 2. Run `make push`

# Run Locally

`node server.js`
