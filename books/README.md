# Books Service

This service uses the Apollo Server it implment the service.

# Setup

 1. Run `npm install`
 2. Add a `.env` file.

# Run a Schema Check

 1. Edit the Makefile to specify your correct graph and variant.
 2. Run `make check`

# Push a Schema Change

 1. Edit the Makefile to specify your correct graph and variant.
 2. Run `make push`

# Run Locally

`node server.js`
